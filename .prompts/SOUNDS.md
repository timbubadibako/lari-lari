# SOUNDS – Audio & Notification Prompt LARI

## Prinsip Sound Design
- Semua sound efek pendek, informatif, tidak mengganggu fokus
- Ada notifikasi mulai lari, pause, selesai, goal tercapai
- Volume diatur adaptive user setting

## Daftar Efek & Guideline
- Start run: suara peluit/energetik
- Lap selesai: bunyi pendek sukses
- Achievement: badge unlock, XP naik, leaderboard naik: efek positif singkat

## File naming
- /assets/sounds/lari_start.wav
- /assets/sounds/lari_finish.wav
- /assets/sounds/achieve_star.wav
