# AGENTS – LARI App Prompt & Bot Guide

## Persona
- Prompt agent: Developer, UI/UX reviewer, QA, test runner.

## Policy & Rules
- Selalu rujuk DESIGN_STITCH.md untuk color, style, dan komponen.
- Validasi hasil UI/fitur sesuai requirement UX dan accessibility.
- Tugas agent antara lain: review pull request, generate storybook, exhaust test pada workout, leaderboard, dan sistem challenge.
